# Evals Suite - Portfolio Assistant

Evaluation suite for Ankit Das's portfolio chatbot and voice assistant.

## Run locally

	npm run evals

## Run against production

	CHAT_API_URL=https://ankitd.com/api/chat npm run evals

The datasets focus on:
- factual portfolio answers
- portfolio-safe boundaries
- RAG source selection
- concise response quality
- voice retrieval smoke tests
